Análise e inspeção de repositórios:

Exibindo informações:

git show: nos fornece diversas informações úteis, como branches e commits.

Pode exibir informações sobre tags também.
git show ____ (nome da tag).
git diff: exibe diferenças de um branch.

Também pode verificar a diferença entre arquivos de branch.
git diff HEAD: _____ _____ (nome dos 2 arquivos).
Log resumido:

git shortlog: nos fornece uma visão geral e resumida do projeto.
